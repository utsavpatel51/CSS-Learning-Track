Example 2 (flex-grow)

Total width is 1536px available. on screen. We're adding 6 item each having 5rem(5*16px=80px) width. Which means without any flex-grow all item will take 480px. So container have 1056px(1536px-480px) free up space. we have set item1 and item 4 grow-factor as 2 and other have grow-factor of 1 means total we have 8 grow-factor, means 132px(1056px/8) more will be added to base width of grow-factor 1 items. for grow-factor 2 it would be 264px more(132px*2). so now item1 and item 4 width will be 344px(264px+80px), all other item width will be 212px(132px+80px).
